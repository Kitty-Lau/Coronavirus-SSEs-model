Folder “figure 4” contains the codes, the results, and the parameters used to retrospectively estimate the size of the South Korea Clusters.  

Please run the codes in the following order:
1. “SouthKorea_MERS.py” runs the estimation using the data in “data_SouthKorea_MERS.csv” and the prior information in “incub_prior_MERS.csv” and “delay_prior_MERS.csv”. It produces the results in folder “mcmc_result_SouthKorea”. 
2. “plot_ SouthKorea.py” plots Figure 4.
